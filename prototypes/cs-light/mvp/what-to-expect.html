What to expect

You can apply now for Countryside Stewardship agreements that start in January 2016.

This is a competitive scheme, so not all applications will be successful.

You can still apply for Countryside Stewardship if you already claim the Basic Payment Scheme (BPS).

  • you'll need to tell us which land parcels you're going to use for the application
  • you can select land management activities from a list of options
  • the options you select will need to be linked to land parcels
  • you might need to draw on the map, or tell us where the land management options will go
  • you can submit the application online, but the agreement will only be offered once we've reviewed it
  • we might contact you before we make the agreement offer
  • once you have our offer, we'll ask you to accept or reject it online
